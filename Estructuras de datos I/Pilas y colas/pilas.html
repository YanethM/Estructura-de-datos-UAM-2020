Pilas
TAD LIFO


Caracteristica Descripción
Descripción Las pilas son una TAD que almacena datos de forma lineal y unidireccional
Los elementos se agregan en un orden y en una sola dirección de inicio a fin.
Siempre el último elemento agregado es el primero en salir.
Historial del navegador web Cada página nueva que se visita se situa encima de la anterior, en el historial de búsqueda. Generando una pila de registros.
Listar información cronológica del más al menos reciente Listar noticias de Instagram, facebook, twitter, todas las redes sociales.
Complejidad en cuanto a tiempo y espacio La complejidad la determina el tipo de implementación si se utiliza un array o objeto
Complejidad tiempo array Acceso: O(1)
Búsqueda: O(n)
Inserción: O(n)
Eliminación: O(n)
Complejidad tiempo objeto Acceso: O(n)
Búsqueda: O(n)
Inserción: O(1)
Eliminación: O(1)
Métodos de las pilas Push: Agregar nuevo valor a la pila, ubicandolo al final de ésta.
Pop: Retorna el último valor ingresado a la pila, eliminandolo de ésta.
Peek: Retorna el último valor ingresado a la pila, sin eliminarlo.
Size: Retorna el número de elementos que contiene la pila
Print: Muestra el contenido de la pila.
